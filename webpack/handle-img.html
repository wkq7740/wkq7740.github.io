# 图片处理

# file-loader

将图片打包拷贝到指定位置

# url-loader

默认将图片解析成 base64 格式，插入打包的 js 代码中。

或者也可以通过设置 limit 属性，将超过一定大小的图片使用 file-loader 来处理。






 




{
    test: /\.(png|svg|gif|jpe?g)$/,
    use: [
            {
            loader: "url-loader",
            limit: 20 * 1024,
            },
    ],
}

# asset

webpack5 新增

配置 所用功能
asset/resource file-loader
asset/inline url-loader
asset/source raw-loader
asset 通过 options 灵活配置

相关配置

modules.exports.output.assetModuleFilename

统一设置 asset 打包处理的文件的输出路径和名称，比较少用，一般是各模块中分开配置

generator




 
 
 


{
    test: /\.(png|svg|gif|jpe?g)$/,
    type: 'asset/resource',
    generator: {
        filename: "img/[name].[hash:4][ext]",   // [ext] 之前不需要.
    }
}

Example (file-loader + url-loader)

{
    test: /\.(png|svg|gif|jpe?g)$/,
    type: 'asset',
    generator: {
        filename: "img/[name].[hash:4][ext]",
    },
    parser: {
        dataUrlCondition: {
            maxSize: 20 * 1024,
        }
    }
}

# asset 处理图标字体

{
    test: /\.(ttf|woff2?)$/,
    type: 'asset/resource',
    generator: {
        filename: "font/[name].[hash:4][ext]",
    },
}
