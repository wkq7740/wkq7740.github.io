# webpack 中的插件

# 简介

与 loader 的区别

名称 功能
loader 用来转换特定类型的模块
plugin 更多，更灵活的功能，比如压缩等

# clean-webpack-plugin

清除 build 目录

# html-webpack-plugin

docs (opens new window)

plugins: [
  new HtmlWebpackPlugin({
    title: "html webpack plugin",
    template: "./public/index.html",
  }),
];

# DefinePlugin

const HtmlWebpackPlugin = require('html-webpack-plugin');
const { DefinePlugin } = require('webpack');

...

plugins: [
  new HtmlWebpackPlugin({
    title: "html webpack plugin",
    template: "./public/index.html",
  }),
  new DefinePlugin({
      BASE_URL: '"./"',
  })
];

# copy-webpack-plugin

复制 public 文件夹中的资源文件，如 favicon.ico 等

new CopyWebpackPlugin({
  patterns: [
    {
      from: 'public',
      globOptions: {
        ignore: ['**/index.html']
      }
    }
  ]
})
