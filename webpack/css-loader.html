# 样式处理

# postcss

用途

帮助我们使用 javascript 来转换样式，如适配浏览器前缀

依赖

browserslist autoprefixer

命令行示例

npx postcss --use autoprefixer -o out.css ./css/exmaple.css

在 webpack 中使用

通过 postcss-loader， 结合插件来使用。例如 autoprefixer，这个插件可以根据 browserslist 获取的浏览器适配列表，为样式加上前缀。

{
    test: /\.css$/,
    use: [
        "style-loader",
        "css-loader",
        {
        loader: "postcss-loader",
        options: {
            postcssOptions: {
            plugins: [
                require("autoprefixer"),
                require("postcss-preset-env"),
            ],
            },
        },
        },
    ],
}

TIP

postcss-preset-env

插件集合，包含更多常见功能，例如将 #12345678 转化为 rgba(18,52,86,0.47059)，其中也包含 autoprefixer

# plugin 配置文件化

以上面的postcss-loader为例，可以在项目目录下新建一个 postcss.config.js 文件，用来指定 postcss-loader 使用到的插件

module.exports = {
  plugins: [require("postcss-preset-env")],
};

然后 webpack.config.js 中的配置就可以简化成：

const path = require("path");

module.exports = {
  entry: "./src/index.js",
  output: {
    filename: "main.js",
    path: path.resolve(__dirname, "dist"),
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ["style-loader", "css-loader", "postcss-loader"],
      },
      {
        test: /\.less$/,
        use: ["style-loader", "css-loader", "postcss-loader", "less-loader"],
      },
    ],
  },
};

# importLoaders

假设有如下两个css文件

login.css

@import "./test.css";

.title {
  color: #12345678;
}

test.css

.title {
  user-select: none;
  transition: all 0.5s;
}

那么在webpack打包时，test.css 中的样式没有被预处理，这是因为 postcss-loader 没有处理 import 语句的功能， 所以它只能处理 login.css 中原有的样式， @import "./test.css"; 则是交给了 css-loader 来处理， css-loader 引入 test.css 后，已经无法反向让 post-loader 处理其中的样式了。

为了解决这个问题，需要用到 importLoaders 属性

const path = require("path");

module.exports = {
  entry: "./src/index.js",
  output: {
    filename: "main.js",
    path: path.resolve(__dirname, "dist"),
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [
          "style-loader",
          {
            loader: "css-loader",
            options: {
              importLoaders: 1,
            },
          },
          "postcss-loader",
        ],
      },
      {
        test: /\.less$/,
        use: ["style-loader", "css-loader", "postcss-loader", "less-loader"],
      },
    ],
  },
};

css-loader 中的 importLoaders 设为1，可以在 import 文件之后，反向回溯1个 loader，让其先处理，再回到 css-loader。
