# webpack-dev-server

# 运行命令

webpack serve --config webpack.config.js

# HMR

webpack.config.js

...

target: 'web',
devServer: {
  hot: true
}

index.js

import "./title";

if (module.hot) {
  module.hot.accept(["./title"], () => {
    console.log("title.js 更新了");
  });
}

# devServer 常用配置

static 设置没有被 webpack 打包的资源的目录 (旧版本是 contentBase)

port 端口号

compress 对打包的 js 文件进行 gzip 压缩

historyApiFallback 将任意的 404 响应替换为 index.html, SPA 中设置为 true

proxy 反向代理
